## 02. lecture indicator dashboard

- absolute vs relative

  - absolute : `x-axis` === 0
    - bar chart
    - bullet chart
    - lollipop chart
    - area chart
  - relative : `x-axis` != 0
    - dot plot
    - box plot
    - line chart

- static vs interactive
  - considering interative dashboard
  - static : `pdf` via email, printing
  - interactive : `dropdown` or `onMouseOver` via web
